### 玩家水平 & pp 相关

- https://syrin.me/pp+/ - PerformancePlus（PP+），一种全新的 pp 算法，网站上可以查询玩家 pp 和排名
- https://ameobea.me/osutrack/ - 个人数据查询网站，用图表形式展示玩家的成长曲线
- http://osuskills.tk/ - osu! 个人实力评测，分 8 项能力进行评分
- http://osustats.ppy.sh/ - 查询玩家谱面排名最高的成绩
